# Utility Commands

Utility commands are available to players at various clearance levels.

| Command | Description | Min. Clearance |
|---------|-------------|----------------|
| `/ac-home [set/delete/list]` | Manage home teleport points | L1 |
| `/ac-tpr` | Teleport to a random location | L1 |
| `/ac-invsee <player>` | View another player's inventory | L3 |
| `/ac-pvp` | Toggle PvP for yourself | L1 |
| `/ac-channels [join/leave] <name>` | Manage chat channels | L1 |
| `/ac-rank <player> <rank>` | Set a player's display rank | L3 |
| `/ac-debug-db [table]` | Debug database contents (admin diagnostic) | L4 |
| `/ac-gui` | Open the in-game GUI menu | L2 |
| `/ac-about` | Show plugin version and info | Anyone |
