# Module Toggle Commands

Each detection module has a dedicated toggle command. These are shortcuts for `/ac-modules <name> <on/off>`.

| Command | Module |
|---------|--------|
| `/ac-fly` | Fly Detection |
| `/ac-killaura` | KillAura Detection |
| `/ac-reach` | Reach Detection |
| `/ac-autoclicker` | AutoClicker Detection |
| `/ac-scaffold` | Scaffold Detection |
| `/ac-xray` | X-Ray Detection |
| `/ac-gamemode` | GameMode Guard |
| `/ac-namespoof` | NameSpoof Detection |
| `/ac-afk` | AFK Detection |
| `/ac-vision` | Vision / Aimbot |
| `/ac-worldborder` | World Border |
| `/ac-lagclear` | Lag Clear |
| `/ac-ratelimit` | Rate Limiter |
| `/ac-packetmonitor` | Packet Monitor |

## Setting Sensitivity

```
/ac-modules <module> sensitivity <1-10>
```

Example:
```
/ac-modules killaura sensitivity 7
```
